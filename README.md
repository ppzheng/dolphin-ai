# Dolphin AI 🐬

**Perceive. Think. Decide. Execute.**

Dolphin AI is a visual strategy builder for AI-driven trading.  
Design, connect, and execute intelligent modules — from perception to execution — in a unified canvas.

---

## ✨ Features

- 🧠 Modular AI system (Sentiment / Social / Market / Strategy / Execution)
- 🎯 Visual strategy builder (Canvas-based)
- ⚡ Real-time signal orchestration
- 🔗 End-to-end decision pipeline

---

## 🧩 Strategy Flow

Perceive → Think → Decide → Execute
